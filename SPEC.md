```python
ai_risk_25 = df25[["q39_4", "q39_5", "q39_6"]].mean(axis=1)
```

2023년에는 오보 리스크 문항이 없으므로, 저작권과 일자리 리스크 문항만 사용한다.

```python

```

연도 간 비교에서는 2023·2025 공통 문항인 저작권·일자리 리스크를 기준으로 비교한다.

### 4.4 역할 실행 갭

역할 실행 갭은 언론 역할 중요도 평균에서 역할 실행도 평균을 뺀 값이다. 값이 클수록 중요하다고 생각하는 역할과 실제 실행 수준 사이의 간극이 크다는 의미다.

```python
role_imp = df[[f"q2_{i}" for i in range(1, 8)]].mean(axis=1)
role_perf = df[[f"q3_{i}" for i in range(1, 8)]].mean(axis=1)
role_gap = role_imp - role_perf
```

### 4.5 번아웃 점수

번아웃 점수는 탈진과 직업 회의 문항의 평균으로 구성한다.

```python
burnout_23 = df23[["q26_3", "q26_4"]].mean(axis=1)
burnout_25 = df25[["q21_3", "q21_4"]].mean(axis=1)
```

## 5. JTSI — 저널리즘 전환 스트레스 지수

### 5.1 정의

JTSI(Journalism Transformation Stress Index)는 디지털 전환 과정에서 언론인이 경험하는 누적 부담을 하나의 지표로 표현하기 위해 구성한 복합 지수다.

본 분석의 JTSI는 2025년 데이터를 기준으로 산출한다.

### 5.2 구성 요소

| 요소        | 변수                | 의미                     |
| --------- | ----------------- | ---------------------- |
| 디지털 피로    | `digital_fatigue` | 디지털 대응과 혁신에 대한 피로감     |
| AI 리스크 인식 | `ai_risk`         | AI 활용으로 발생할 수 있는 위험 인식 |
| 역할 실행 갭   | `role_gap`        | 저널리즘 역할의 중요도와 실행도 간 차이 |
| 번아웃       | `burnout`         | 업무 탈진과 직업 회의           |

### 5.3 산식

서로 다른 척도를 가진 변수를 합산하기 위해 각 구성 요소를 z-score로 표준화한 뒤 합산한다.

```text
JTSI_i = z(디지털 피로_i) + z(AI 리스크_i) + z(역할 실행 갭_i) + z(번아웃_i)
```

```python
from scipy.stats import zscore

components = ["digital_fatigue", "ai_risk", "role_gap", "burnout"]
df_jtsi = df.dropna(subset=components).copy()

for col in components:
    df_jtsi[f"{col}_z"] = zscore(df_jtsi[col])

df_jtsi["JTSI"] = df_jtsi[[f"{col}_z" for col in components]].sum(axis=1)
```

### 5.4 해석

* JTSI가 0보다 크면 평균보다 전환 스트레스가 높은 집단으로 해석한다.
* JTSI가 0보다 작으면 평균보다 전환 스트레스가 낮은 집단으로 해석한다.
* JTSI는 개인의 심리 상태를 진단하는 임상 지표가 아니라, 집단별 전환 부담을 비교하기 위한 분석 지표다.

2025년 기준 검산값:

| 항목         |             값 |
| ---------- | ------------: |
| JTSI 범위    | -8.50 ~ +8.70 |
| 전체 평균      |           약 0 |
| AI 활용자 평균  |        +0.029 |
| AI 비활용자 평균 |        -0.040 |

### STEP 5.5. JTSI 민감도 검토

JTSI 구성 방식에 따라 결론이 얼마나 달라지는지 확인하기 위해 세 가지 버전을 비교한다.

| 버전      | 구성                       | 성격    |
| ------- | ------------------------ | ----- |
| JTSI v1 | 피로 + AI 리스크 + 역할 갭 + 번아웃 | 기본 모델 |

| JTSI v3 | v1 + 직무 불만족              | 확장 모델 |

주요 해석:

* 인터넷언론사는 세 버전 모두에서 가장 낮은 JTSI를 보였다.
* 전통매체 간 세부 순위는 구성에 따라 달라졌다.
* 따라서 개별 매체 순위보다 인터넷언론사와 전통매체군의 큰 구도를 중심으로 해석한다.

산출물:

```text
notebooks/05_5_sensitivity.ipynb
outputs/tables/jtsi_sensitivity.csv
```

### STEP 6. 통계적 타당성 확인

집단 간 차이와 범주형 변수 간 연관을 검정한다.

| 분석            | 방법           | 효과크기        |
| ------------- | ------------ | ----------- |
| 두 집단 평균 차이    | Welch t-test | Cohen’s d   |
| 세 집단 이상 평균 차이 | ANOVA        | eta squared |
| 범주형 변수 연관     | 카이제곱 검정      | Cramér’s V  |

효과크기 기준:

| 효과크기       |   작음 |   중간 |    큼 |
| ---------- | ---: | ---: | ---: |
| Cohen’s d  |  0.2 |  0.5 |  0.8 |
| eta²       | 0.01 | 0.06 | 0.14 |
| Cramér’s V |  0.1 |  0.3 |  0.5 |

주요 결과:

| 검정              |     p값 |       효과크기 | 해석            |
| --------------- | -----: | ---------: | ------------- |
| 매체유형별 JTSI 차이   | p<.001 | eta²=0.008 | 유의하지만 효과크기 작음 |
| 매체유형 × AI 활용 여부 | p<.001 |    V=0.092 | 유의하지만 효과크기 작음 |

산출물:


### STEP 6.5. 2×2 위기집단 진단

## 8. 분석 재현 시 유의사항

1. 본 분석은 SAV 파일을 기준으로 재현한다.
2. 2023년과 2025년 일부 문항 번호가 다르므로 변수 매핑을 확인해야 한다.
3. AI 리스크의 연도 비교는 공통 문항인 저작권·일자리 리스크를 기준으로 한다.
4. JTSI는 z-score 표준화 후 합산한다.
5. JTSI는 집단 간 전환 부담 비교를 위한 분석 지표이며, 개인 수준의 임상적 진단 지표로 해석하지 않는다.
6. 통계적으로 유의하더라도 효과크기가 작은 결과는 강한 차이로 해석하지 않는다.
7. 원본 데이터 파일은 GitHub에 업로드하지 않는다.
8. 정책 제안은 분석 결과에서 확인된 수치를 근거로 해석한다.
